Utilities to allow clients to export and import handles to a surface.

This module provides automatic management of exporting surfaces from one client and allowing another client to import the surface using a handle. Management of the lifetimes of exports, imports and allowing imported surfaces to be set as the parent of a surface are handled by the module.

How to use it

Having a functional xdg_shell global setup is required.

With a valid xdg_shell global, simply initialize the xdg_foreign global using the xdg_foreign_init function. After that just ensure the XdgForeignState returned by the function is kept alive.


// XDG Shell init
let (shell_state, _) = xdg_shell_init(
    &mut display,
    |event: XdgRequest, dispatch_data| { /* handle the shell requests here */ },
    None
);

let (foreign_state, _, _) = xdg_foreign_init(&mut display, shell_state.clone(), None);

// Good to go!
