Utilities for handling shell surfaces with the wlr_layer_shell protocol

This interface should be suitable for the implementation of many desktop shell components, and a broad number of other applications that interact with the desktop.

Initialization

To initialize this handler, simple use the wlr_layer_shell_init function provided in this module. You need to provide a closure that will be invoked whenever some action is required from you, are represented by the LayerShellRequest enum.

use smithay::wayland::shell::wlr_layer::{wlr_layer_shell_init, LayerShellRequest};

let (shell_state, _) = wlr_layer_shell_init(
    &mut display,
    // your implementation
    |event: LayerShellRequest, dispatch_data| { /* handle the shell requests here */ },
    None  // put a logger if you want
);

// You're now ready to go!
